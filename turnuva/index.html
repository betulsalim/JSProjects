JSON means: JavaScript Object Notation.

  • JSON is a data interchange format.
  • Compare to XML: JSON carries data in smaller size.
  • Compare to XML: JSON works more fast and because of the difference write style JSON takes less place


JSON Areas Of Use:

  • "JavaScript Object Notation" is designed for JavaScript but today we use JSON with lots of coding language and platforms:
    1. Web sevices, mobile applications, PHP, .NET, Java...
  • Data can be moved between mobile applications and desktop software with JSON


JSON Data Types:

  • String
  • Numbers
  • Boolean
  • Array
  • Object
  • Null
